# Kitchen Casework — Project Specification

**Designer:** Callie Wells, Interior Design
**Project:** Single-wall Kitchen with Island
**Location:** Rancho Santa Margarita, CA
**Sheet set:** A-201 Plan, Elevations & 3D
**Scale:** 1/2" = 1'-0" (plan)
**Date:** 2026-04-27
**Drawing file:** `kitchen.dwg`

---

## 1. Project narrative

A clean single-wall kitchen for a coastal-California home — long-wall
casework for everyday cooking, anchored by a freestanding island that
doubles as the breakfast bar and the prep zone. Painted Shaker doors in a
warm white over a quartz counter keep the room bright; a tall pantry on
the east end disguises the dry-goods storage so the line of upper cabinets
reads continuous from the dining room (visible through the west-wall
opening).

**Design intent**

- **One wall, full work triangle.** Range, sink, and refrigerator-adjacent
  zone (the dishwasher slot) all live along the south wall in a 14'-0"
  run. Total prep travel is under 6'-0" between work zones.
- **Island as second counter.** The island carries the seating overhang
  (12" on the north side) and provides 96" of clear counter for prep —
  enough for a sheet pan and two cutting boards laid out at once.
- **Pantry over wall cabinets.** The east-end pantry runs full 84" so it
  reads as architecture, not as cabinetry. Wall cabinets stop short of
  it for visual rhythm.
- **Hood over the range.** No wall cabinet over the cooktop — a 30"
  ducted hood vented up through the soffit. Code-compliant, and visually
  it gives the cooktop "breathing room."

This sheet set focuses on casework. A reflected-ceiling plan is a
recommended add-on:

- 4× recessed 4" LED downlights in a staggered grid over the island and
  prep run (3000K, 800 lm, dimmable)
- Under-cabinet LED strip below all wall cabinets (2700K, switched at
  the island)
- Pendant-light cluster (3 pendants) over the island, hung at 36" above
  the counter
- All zones on Lutron Caseta dimmers, scene control by the dining-room
  doorway

## 8. 3D modeling notes

The .dwg uses the brief's Project 4 commands:

| Command | Use |
| --- | --- |
| `BOX` | All cabinet boxes, counter, hood, range, dishwasher, sink basin, pantry |
| `CYLINDER` | Cooktop grates, faucet body and spout |
| `SUBTRACT` | Hallway opening, sink cutout in counter, toe kicks (as inverted SUBTRACT instead of `PRESSPULL`) |
| `2DWireframe` / `Conceptual` | Plan + elevations vs. 3D presentation |
| TOP / NEISO / FRONT / BACK | Plan view + iso + two elevations |

### Items reserved for the GUI / a future pass

- **`UCS` named coordinate systems.** The brief calls for naming a UCS
  per elevation; not done in this script. The elevations were captured
  via the FRONT / BACK named views with a frozen island and a frozen
  cabinet run respectively.
- **`PRESSPULL`** — the toe kicks were cut with `SUBTRACT` (an inverted
  void box) since PRESSPULL is interactive. Same final geometry.
- **`FILLETEDGE` 0.25" on counter edges** — not applied in script
  (FILLETEDGE works via SendCommand+LISP from session 04-25, but counter
  edges read fine without the fillet at this scale).
- **`FLATSHOT` 2D projection from the 3D model** — would give a verified
  elevation drawing overlay on the 2D elevations. Not driven here; the
  elevations in this package are pure 3D-projected views (FRONT/BACK)
  rather than dedicated 2D linework.
- **ARCH-D layout sheet with title block + 4 viewports** (plan,
  Elevation A, Elevation B, isometric) — Word document substitutes for
  the plotted board.

### Run-length adjustment vs. the brief

The original brief calls out a long-wall cabinet run summing to **192"**
(B36 + R30 + B30 + SB36 + B36 + P24) which exceeds the **168"** of a 14'
wall by 24". This package adjusts the run to fit:

- **B36 + R30 + DW24 + SB36 + B30 + P12 = 168"**

The `B30` between range and sink-base in the brief is reassigned to a
24" dishwasher slot (which the brief lists separately under
appliances), and the pantry is reduced from 24" to 12" to fit the wall.
A 12" pantry is suitable for canned goods + spices; a deeper walk-in
pantry would belong in an adjacent room.
