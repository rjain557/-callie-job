## Gotchas discovered

- The repo root (`-callie-job/`) starts with a dash — `mv` treats it as a flag unless you use `./` prefix or `--` separator.
- `.mcp.json` already existed in the repo (for `googleworkspace`). Always read before overwriting shared config files.
- Unicode characters (em-dash, smart quotes) in `.ps1` files break the parser in some encodings. Stick to ASCII in PowerShell scripts.
